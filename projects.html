CrimeFace

A cutting-edge face recognition app designed to enhance public safety. CrimeFace allows users to capture photos, compare facial features with a criminal database, and instantly retrieve matching profiles. Equipped with secure authentication and a streamlined interface, it simplifies data management and ensures accurate, reliable results for crime prevention efforts.

NAIL

NAIL - No Access Inhibitor Latch is an innovative traffic control system designed to enhance road safety by enforcing red light compliance. Using hydraulic mechanisms, this system prevents vehicles from breaking the signal when it turns red, halting movement until the light changes. NAIL aims to reduce traffic violations, improve traffic flow, and decrease accidents at intersections by ensuring vehicles cannot bypass red lights, even in cases of driver negligence or impatience.

PROBE

PROBE - Proactive Real-Time Observation of Water’s Bio-Physical Environment is an advanced system for monitoring and analyzing water environments in real time. It collects critical bio-physical data, such as temperature, pH, oxygen levels, and other vital environmental parameters, to ensure sustainable water management. Designed for researchers, environmentalists, and water quality managers, PROBE helps in proactive decision-making to maintain water ecosystem health and prevent pollution.

Bus.io

A console-based C program that implements a dynamic bus reservation system using a Binary Search Tree (BST) for managing customer information. The system supports features such as login, bus selection, ticket booking, seat availability check, reservation cancellation, and retrieval of reservation details. It also includes error handling, dynamic memory allocation, and a simple user interface with colored outputs to enhance user experience.
